Adicionar Bibliotecas ao Projeto:
*No NetBeans, no projeto, clicar com botão direito em 'Bibliotecas'
*Selecionar 'Adicional biblioteca...' -> 'Criar...'

	JasperReports:
	*Nomear como 'JasperReports 6.8.0'
	
	PostgreSQL:
	*Nomear como 'PostgreSQL 42.2.5'
	
*Clicar em 'Adicionar Jar/Pasta...'
*Selecionar '.jar' dentro da pasta 'libs' do Projeto
*Clicar em 'Ok' e, com a biblioteca selecionado, clicar em 'Adicionar Biblioteca'
